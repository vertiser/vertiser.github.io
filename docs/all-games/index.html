All Games project

The All Games project is my personal commitment to play every single game that I own. I have dozens of games in various game libraries (Steam, GOG, Nintendo Switch, etc.), many of which I have never installed, or have only opened briefly. I want to immerse myself in new titles that are already in my library, waiting for discovery. The current page is used to track progress of my challenge. To see a list of all articles related to the project follow this link.

Last updated: 09.01.2024

Rules

  • each game should be experienced in a meaningful way (no playing for 5 minutes to call it ‘done’)
  • no requirement to complete the main quest line, campaign or to get all the trophies/achievements, etc.
  • games I don’t like can be abandoned after several attempts to find some likeable elements
  • games already completed are marked as such and backdated
  • if I own game on multiple platforms it only needs to be completed on one
  • after a new game is bought it will be added to the list and the overall progress will be adjusted
  • there is no deadline, schedule or expected pace
